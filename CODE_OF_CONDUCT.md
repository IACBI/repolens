# Code of Conduct

RepoLens aims to be a respectful, practical, and welcoming open-source project.

## Expected Behavior

- Be respectful and constructive.
- Focus criticism on code, design, and project outcomes.
- Assume good intent while still being clear about technical concerns.
- Respect privacy and avoid sharing sensitive repository details.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Repeated disruption of project discussions.

## Enforcement

The maintainer may remove comments, close issues, reject contributions, or block participants who violate this code of conduct.

## Reporting

If a report involves a security issue, follow SECURITY.md. For conduct concerns, contact the maintainer through the repository's GitHub moderation or issue-management tools after the repository is hosted.
