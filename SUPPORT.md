# Support

RepoLens is an open-source project.

## Getting Help

- Read README.md and the docs directory first.
- Use `pnpm dev -- summary` to confirm RepoLens can analyze the current repository.
- Use `pnpm dev -- check-docs` to inspect documentation health warnings.

## Questions

After the GitHub repository is hosted, use the question issue template for usage questions. Include your Node.js version, pnpm version, operating system, command used, and relevant terminal output.

## Bugs

Use the bug report template. Include a minimal reproduction repository shape when possible. Do not include private source code unless you are comfortable publishing it.
