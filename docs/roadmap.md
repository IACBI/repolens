# Roadmap

[English](roadmap.md) | [Türkçe](roadmap.tr.md)

RepoLens should stay focused on local-first repository understanding.

## Near Term

- Improve JavaScript/TypeScript import parsing.
- Add more edge-case tests for documentation health checks.
- Improve monorepo module detection.
- Add stricter CI options for documentation health.

## Later

- Add opt-in provider integrations behind explicit config.
- Improve Go and Rust dependency detection.
- Add richer output customization.
- Add release automation after the project is hosted.

## Out of Scope for the MVP

- Web dashboard
- Authentication
- Cloud sync
- External AI calls by default
- Embeddings or vector databases
- Editor extensions
